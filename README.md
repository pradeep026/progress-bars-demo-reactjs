## Prerequisites

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM)
* [Bower](http://bower.io/)


## Installation

* `git clone <repository-url>` this repository
*  cd into the new directory
* `npm install`
* `bower install`

## Running the application

* `npm start`
* Visit your app at [http://localhost:3000](http://localhost:3000)

## Unit Test

* `npm test`
